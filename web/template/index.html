Covid19-Kerala-Api

A fast and simple API service for tracking Coronavirus(COVID-19, SARS-CoV-2) outbreak in Kerala(India).

Id Endpoint/Route Type Description
1 /api Object Provides all the available data which includes district wise and total data on number of postive cases, no of persons hospitalized, inclusion from other districts etc
2 /api/location Array Provides an array of all the districts in the same format as the key in 'all' object
3 /api/location?loc=ernakulam&loc=kannur Object Provides a single object with the details corresponding to each of the district mentioned in loc=xxx
