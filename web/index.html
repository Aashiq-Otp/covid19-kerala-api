Covid19-Kerala-Api

A fast and simple API service for tracking Coronavirus(COVID-19, SARS-CoV-2) outbreak in Kerala(India).

Id Endpoint/Route Type Description
1 /api Object Provides all the available data which includes district wise and total data on number of positive cases, no of persons hospitalized, inclusion from other districts etc
1 /api/timeline Object Provides a timeline of the no of positive cases from start till latest date[WIP]
2 /api/location Array Provides an array of all available location values/districts which are acceptable by 'loc' param
3 /api/location?date=<01-04-2020< /a> Object Provides an object filtered out with only the specified dates(dd-mm-yyyy or dd/mm/yyyy) - supports '<','>' filters along with date param. User can even give 'date=latest' to get latest data only.
3 /api/location?loc=ernakulam&loc=kannur Object Provides an object with details filtered out with only the specified locations(loc)
